# Codificador
Codificador de textos - ONE Challenge

Neste codificador foi implementado breves conhecimentos básicos de HTML, CSS e Javascript. Com uma lógica de codificação bem estruturada. 
Foi realizado com base em alguns requisitos propostos pela Alura no seu programa ONE, onde é apresentado um desafio para os alunos treinarem a sua capacidade de criar programas funcionais.
